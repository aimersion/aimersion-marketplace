Get Started in 60 Seconds

Everything you need to start using Aimersion plugins with Claude Code

1

Install Claude Code

Claude Code is Anthropic's CLI for agentic coding. Install it globally with npm, then authenticate.

npm install -g @anthropic-ai/claude-code
2

Add This Marketplace

Add the Aimersion marketplace with one command. All 72 plugins become available to browse and install.

/plugin marketplace add aimersion/aimersion-marketplace
3

Install a Plugin

Click any plugin card below to see its skills and commands, then copy the install command.

/plugin install aimersion/aimersion-marketplace/[name]
4

Use Skills & Commands

Each plugin adds specialized skills and slash commands directly into Claude Code. Type / to see them.
