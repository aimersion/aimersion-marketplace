Getting Started

Up and running in minutes

Choose your platform — each has a direct path to using any plugin.

1

Go to Claude.ai

Open claude.ai and create a new Project — or open an existing one. Projects give Claude persistent instructions for every chat.

2

Find a Plugin

Browse the catalog below. Click any plugin card to see its skills, then click Copy for Claude.ai to copy the skill content.

3

Paste into Instructions

In your Project, click Edit Instructions and paste. Claude now has that skill for every conversation in the project.

Project → Edit Instructions → Paste
4

Just Chat

No slash commands needed. Describe your task naturally — the skill activates automatically based on what you ask.

1

Install Claude Code

Requires Node.js 18+. Install the CLI globally and authenticate with your Anthropic account.

npm install -g @anthropic-ai/claude-code
2

Go to Your Project

Navigate to the directory where you want the plugin active. Claude Code reads CLAUDE.md automatically from the current directory.

cd ~/your-project
3

Run the Installer

Click a plugin below, copy the install command from the Claude Code tab, and run it. Skills download into your CLAUDE.md.

curl -sL [install-url] | bash pool-service-operator
4

Start Claude Code

Run claude in the directory. Skills activate immediately. Type / to see available slash commands from the plugin.

claude
1

Pick a Skill

Each skill is a standalone SKILL.md file. Browse the catalog and click any plugin to find the raw GitHub URL for the skills you want.

2

Fetch the File

Pull the raw SKILL.md at runtime or cache it. No parsing required — it's plain markdown designed for system prompts.

curl .../plugins/[name]/skills/[skill]/SKILL.md
3

Inject into System Prompt

Pass the skill content in the system parameter of your API call alongside your own instructions.

system: f"You are a [role].\n\n{skill_content}"
4

Stack Multiple Skills

Combine skills from different plugins in one system prompt to build compound agents. Skills are composable by design.
