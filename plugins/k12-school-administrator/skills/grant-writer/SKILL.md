# Grant Writer

Draft compelling grant applications, budget narratives, and supporting documents for K-12 education funding — federal (Title I, IDEA, ESSER), state, and private foundation grants.

## Grant Application Structure

### 1. Executive Summary / Abstract
- One page maximum
- State the problem, the proposed solution, the target population, and expected outcomes
- Include the total funding request and project duration
- Write this LAST but place it FIRST in the application

### 2. Statement of Need
- Use local data: demographics, achievement gaps, poverty rates, attendance data
- Compare school/district data to state and national benchmarks
- Cite recent, authoritative sources (NCES, state education department)
- Tell a story with data — connect statistics to real student experiences
- Quantify the gap between current state and desired outcomes

### 3. Project Description / Plan of Action
- Clearly describe what the grant funds will support
- Align activities to evidence-based practices (cite ESSA evidence tiers)
- Include a detailed implementation timeline with milestones
- Describe staffing, roles, and responsibilities
- Explain how the project aligns with school/district strategic plan

### 4. Goals, Objectives, and Outcomes
- **Goals**: Broad, aspirational statements of impact
- **Objectives**: SMART (Specific, Measurable, Achievable, Relevant, Time-bound)
- **Outcomes**: Both quantitative (test scores, graduation rates) and qualitative (surveys, observations)
- Include a logic model connecting inputs → activities → outputs → outcomes

### 5. Evaluation Plan
- Describe formative and summative evaluation methods
- Identify data sources and collection timeline
- Name the evaluator (internal or external)
- Explain how data will inform program adjustments

### 6. Budget and Budget Narrative
- Line-item budget aligned to project activities
- Budget narrative justifying every expense
- Show cost-effectiveness and sustainability plan
- Include matching funds or in-kind contributions if applicable
- Common categories: Personnel, Fringe Benefits, Travel, Equipment, Supplies, Contractual, Other, Indirect Costs

### 7. Sustainability Plan
- How the program will continue after grant funding ends
- Identify alternative funding sources
- Describe institutional capacity built during the grant period

## Budget Narrative Best Practices

For each line item, explain:
1. **What** — The specific expense
2. **Why** — How it connects to the project goals
3. **How much** — The calculation showing how the amount was derived
4. **When** — The timeline for the expenditure

Example format:
> **Personnel — Reading Specialist (1.0 FTE): $65,000**
> A full-time reading specialist will provide targeted Tier 2 and Tier 3 interventions to 45 students in grades 3-5 identified through universal screening as reading below grade level. Salary is based on the district's certified salary schedule, Step 8, MA+30.

## Common Education Grant Programs

- **Title I, Part A** — Supporting disadvantaged students
- **Title II, Part A** — Teacher quality and professional development
- **Title III** — English Language Learner programs
- **Title IV, Part A** — Student Support and Academic Enrichment
- **IDEA Part B** — Special education services
- **21st Century Community Learning Centers** — Before/after school programs
- **E-Rate** — Technology and internet connectivity
- **Perkins V** — Career and Technical Education

## Writing Tips

- Use active voice and concrete language
- Avoid jargon — write for reviewers who may not be educators
- Address every criterion in the scoring rubric
- Include letters of support from partners and stakeholders
- Proofread for consistency in numbers between narrative and budget
