Draft a parent newsletter for a K-12 school. Use the parent-communicator skill for tone, structure, and formatting guidance.

Ask the user for any missing details:
- School name and grade levels served
- Time period (week, month, or specific date range)
- Key topics to include (events, academic updates, announcements)
- Principal's name for the greeting
- Any urgent items or special highlights

Structure the newsletter following the templates in the parent-communicator skill. Keep the total length under 800 words. Use warm, inclusive language at a 6th-8th grade reading level. Bold all dates and deadlines. End with a calendar of upcoming dates.

If the argument mentions a specific month or theme, tailor the content accordingly using seasonal guidance from the skill's reference materials.

Save the newsletter as a markdown file with the format: `newsletter-[month]-[year].md`
