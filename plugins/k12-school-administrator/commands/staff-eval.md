Create a staff evaluation document using the staff-evaluator skill for frameworks, rubrics, and best practices.

Determine the type of document needed:
1. **Pre-observation form** — Prepare for an upcoming classroom visit
2. **Observation report** — Document a completed classroom observation
3. **Summative evaluation** — End-of-cycle performance evaluation
4. **Professional development plan** — Growth plan for a specific educator
5. **Improvement plan** — Structured support plan for struggling staff

Ask the user for relevant details:
- Teacher name, subject, and grade level
- Observation date and lesson topic (if applicable)
- Specific strengths or concerns to address
- Evidence and examples from the observation
- Rating framework being used (Danielson, Marzano, or district-specific)

Use the Praise-Question-Polish feedback framework. Support every rating with at least 2 specific, observable examples. Frame all feedback as growth-oriented. Reference the observation rubrics in the skill's reference materials for domain-level descriptors.

Save the document with the format: `eval-[teacher-lastname]-[date].md`
