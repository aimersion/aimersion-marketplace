description: Draft an education grant application or budget narrative
allowed-tools: Read, Write, Edit, WebSearch
argument-hint: [grant name or funding need]
---

Draft a grant application or budget narrative for a K-12 school using the grant-writer skill for structure, writing guidance, and budget templates.

Identify the grant type or funding source:
- Federal programs (Title I, Title II, Title III, IDEA, ESSER, 21st CCLC)
- State education department grants
- Private foundation grants
- Corporate education grants

Ask the user for:
- Grant name and funding agency
- Funding amount requested
- Problem or need to address
- Proposed program or intervention
- Target student population and numbers
- School/district demographic data
- Project duration
- Scoring rubric or evaluation criteria (if available)

Draft all required sections: executive summary, statement of need with data, project description, SMART objectives, evaluation plan, budget with line items, budget narrative with calculations, and sustainability plan. Align activities to ESSA evidence tiers where applicable. Use the budget templates from the skill's reference materials.

If the user provides a specific grant's RFP or scoring rubric, tailor the application to address every criterion explicitly.

Save as `grant-[program-name]-[year].md`
