---
description: Design or optimize a customer loyalty program
argument-hint: [program type: membership, points, or referral]
---

Design a complete customer retention and loyalty program for the detailing business.

Ask the user for:
- Current customer base size (approximate)
- Average service frequency of current customers
- Current retention rate (if known)
- Budget for loyalty incentives
- Preferred program type (membership, points, referral, or comprehensive)

Generate a complete program using the customer-retention skill:

1. **Program structure**: Detailed tiers, benefits, and pricing (for membership) or earning/redemption rules (for points)
2. **Financial projections**: Expected enrollment rate, monthly recurring revenue, and ROI
3. **Launch plan**: How to introduce the program to existing customers
4. **Marketing materials**: Enrollment pitch scripts (in-person, text, email), social media announcement posts
5. **Retention communication cadence**: Automated touchpoints to keep customers engaged between visits
6. **Referral component**: How to incentivize word-of-mouth within the loyalty framework

Make the program simple enough to manage without complex software — a spreadsheet or basic CRM should be sufficient for tracking.
