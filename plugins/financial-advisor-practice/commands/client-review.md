Prepare a comprehensive client portfolio review for $ARGUMENTS.

Follow the client-reviewer skill methodology:

1. Ask for any missing information: client name, risk tolerance, investment objectives, time horizon, account types, and current holdings.
2. Generate a complete performance summary table (MTD, QTD, YTD, 1Y, 3Y, 5Y) with benchmark comparison.
3. Create an asset allocation analysis comparing current vs. target with drift flags.
4. Produce rebalancing recommendations with tax implications noted.
5. Write 5-7 talking points for the client meeting.
6. Generate both a client-facing summary and internal advisor prep notes.
7. List action items and follow-up tasks.

Output the review as a well-structured document. Include standard compliance disclaimers about past performance and hypothetical projections.
