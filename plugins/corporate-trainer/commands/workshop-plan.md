Plan an interactive workshop using the workshop-builder skill for agenda design, facilitation techniques, and engagement strategies.

Ask the user for: workshop topic and learning objectives, duration, audience size and composition, delivery format (in-person, virtual, hybrid), participant familiarity with the topic, and desired outcomes.

Deliver a complete workshop plan: timed agenda, facilitation guide with talking points, activity descriptions with instructions, materials list, setup requirements, and a contingency plan for timing adjustments.

Save as `workshop-[topic]-[duration].md`
