Generate a comprehensive security posture assessment for an MSP client.

Gather:
- Client company name and industry
- Number of users and locations
- Current infrastructure overview (on-prem, cloud, hybrid)
- Compliance requirements (HIPAA, PCI-DSS, CMMC, SOC 2, etc.)
- Previous security assessments (if any)
- Known security concerns

Produce a security assessment report:

## 1. Executive Summary
- Overall security posture rating (A-F or 1-100 score)
- Top 5 critical findings
- Recommended immediate actions
- Investment summary for remediation

## 2. Identity and Access Management
- Password policy assessment
- MFA deployment status
- Privileged access management
- Conditional access policies
- Offboarding process review
- Service account inventory

## 3. Endpoint Security
- Antivirus/EDR coverage
- OS patch management status
- Encryption (BitLocker/FileVault) deployment
- BYOD policy and MDM coverage
- USB/removable media controls

## 4. Network Security
- Firewall configuration review
- VPN setup and policies
- Network segmentation
- Wi-Fi security (WPA3, guest isolation)
- DNS filtering status

## 5. Email Security
- SPF, DKIM, DMARC configuration
- Anti-phishing controls
- Email encryption capability
- Attachment filtering rules
- Quarantine management

## 6. Data Protection
- Backup strategy (3-2-1 rule compliance)
- Backup testing frequency
- Data classification policy
- DLP controls
- Retention policies

## 7. Business Continuity
- Disaster recovery plan existence and testing
- RTO/RPO definitions
- Failover capabilities
- Communication plan during incidents

## 8. Compliance Mapping
Map findings against applicable compliance frameworks with gap analysis.

## 9. Remediation Roadmap
Prioritized remediation plan with:
- Immediate (0-30 days): Critical vulnerabilities
- Short-term (30-90 days): High-priority gaps
- Medium-term (90-180 days): Strategic improvements
- Ongoing: Maintenance and monitoring

Save as a professional PDF-ready report.
