# RFE Response Command

Request from the client:

1. **Form type and case number** (e.g., I-485, I-130, N-400)
2. **RFE or NOID content** (Request for Evidence or Notice of Intent to Deny)
3. **Specific deficiencies identified** (list each issue raised)
4. **Available evidence** (what documentation they can provide)
5. **Response deadline date**

## Generate a comprehensive RFE response strategy that includes:

### Response Timeline and Deadlines:

- Days remaining to submit response
- Recommendation for request additional time if needed (Form I-131B or written request explaining good cause)
- Submission deadline accounting for mail delivery time

### Cover Letter Outline:

- Format recommendations (business letter format, on letterhead)
- Required elements (case number, RFE issue date, each issue number referenced)
- Professional tone guidance
- Structure for addressing each RFE deficiency

### Evidence Gathering Strategy:

- For each RFE issue, identify:
  - What evidence is being submitted
  - Why it addresses the deficiency
  - Any explanatory letter needed
  - Supporting documentation strategy

### Addressing Each Deficiency:

- Organize response by RFE issue number
- Explain how each piece of evidence addresses the specific concern
- Note any legal precedent or regulation supporting the response
- Flag issues that may require expert declarations (medical, financial, country conditions)

### Special Considerations:

- Whether the RFE relates to establishment of prima facie case or credibility
- Strategic response considerations (admitting/addressing negative evidence vs. providing positive evidence)
- Whether additional time should be requested
- Any supplemental expert declarations or affidavits needed
