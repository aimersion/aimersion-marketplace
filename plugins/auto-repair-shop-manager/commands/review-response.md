Draft a professional response to the online review provided.

Ask the user for:
- The review text (or summary)
- Star rating
- Platform (Google, Yelp, Facebook, etc.)
- Any context about the customer's visit (optional)

Generate a response following the review-manager skill guidelines:

For **positive reviews** (4-5 stars):
- Thank by name, reference specifics, reinforce shop values, invite return
- Keep it genuine and brief (2-4 sentences)

For **negative reviews** (1-2 stars):
- Acknowledge and empathize — never argue or be defensive
- Apologize for the experience
- Take it offline with a direct phone number
- Show commitment to resolution
- Never reveal private repair details

For **mixed reviews** (3 stars):
- Thank for feedback, acknowledge positives, address concerns, offer to discuss

Provide the response ready to post. Also suggest any follow-up actions the shop should take internally (e.g., investigate the complaint, train staff, update processes).
