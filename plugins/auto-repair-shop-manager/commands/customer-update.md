description: Draft a customer communication message
argument-hint: [update type: diagnosis, authorization, ready, follow-up]
---

Draft a professional customer communication for the specified stage of the repair process.

Ask the user for:
- Communication type (diagnosis results, authorization request, vehicle ready, status update, post-service follow-up)
- Customer name and vehicle
- Key details (findings, cost, timeline, etc.)
- Preferred channel (text, email, or phone script)

Generate the message using the customer-communicator skill guidelines:
- Use plain language — explain what parts do, not just technical names
- Lead with safety if relevant
- Be transparent about costs and timeline
- Include a clear next step or call-to-action

For text messages, keep under 300 characters and include a response prompt.
For emails, use a clear subject line and scannable format.
For phone scripts, include the key talking points and anticipated customer questions.

Make every message ready to send — the user should be able to copy and use it immediately.
