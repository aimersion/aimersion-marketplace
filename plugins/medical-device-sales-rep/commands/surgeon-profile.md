---
description: Build or update a surgeon relationship profile
allowed-tools: Read, Write, Edit, WebSearch
argument-hint: [surgeon name]
---

Use the surgeon-relationship skill to create or update a comprehensive surgeon profile.

1. Ask for the surgeon's name (or use $ARGUMENTS). Search for publicly available information about the surgeon (publications, affiliations, specialty focus).

2. Ask the user to fill in relationship-specific details:
   - Current device preferences and procedure volume
   - Communication preferences and best contact methods
   - Decision-making style (data-driven, peer-influenced, early adopter)
   - Key influencers on their team
   - Competitive products in use
   - Open opportunities or evaluations

3. Generate a surgeon profile document with:
   - Professional summary (specialty, training, publications, affiliations)
   - Relationship status and engagement history
   - Device usage and business summary
   - Engagement strategy with specific next steps
   - Value-add opportunities tailored to this surgeon's interests
   - Follow-up schedule with templates for common touchpoints

4. Save the profile to a document.

All surgeon profiles must comply with AdvaMed Code of Ethics guidelines.
