Use the case-support-prep skill to create a comprehensive case preparation package.

1. Ask the user for:
   - Surgeon name and facility
   - Procedure type and approach
   - Patient-relevant details the surgeon has shared (age, anatomy, revision vs. primary)
   - Device/implant system being used
   - Any special considerations or surgeon preferences

2. Generate a case prep document with:
   - Surgeon preference checklist (based on known preferences or standard setup)
   - Complete inventory list with primary sizes and backup options
   - Step-by-step procedural overview with key rep involvement points
   - Potential intraoperative challenges and contingency plans
   - Post-case documentation requirements (lot numbers, stickers, forms)

3. Include a one-page "cheat sheet" the rep can reference during the case.

4. Save all materials to downloadable documents.

If $ARGUMENTS specify a surgeon or procedure, use that as the starting context.
