Design a complete online course curriculum using the curriculum-designer skill for pedagogy, module sequencing, and assessment strategies.

Ask the user for:
- Course topic and niche
- Target audience (beginners, intermediate, advanced, professionals)
- Desired transformation ("Students will go from ___ to ___")
- Preferred course length (mini, standard, flagship, cohort)
- Content format preferences (video, text, interactive, live)
- Any existing content or materials to incorporate

Deliver a complete course blueprint including: transformation statement, 3-7 course-level learning objectives, module-by-module outline with titles and objectives, content format recommendations per module, assessment strategy for each module, estimated completion times, and prerequisite mapping. Reference the course blueprint templates from the skill's reference materials.

Save as `course-design-[topic].md`
