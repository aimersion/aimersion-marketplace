Write lesson content for an online course using the lesson-writer skill for structure, engagement techniques, and formatting.

Determine the content type needed:
1. **Video script** — Formatted with speaker notes, B-roll cues, and visual overlays
2. **Text lesson** — Structured with headers, examples, and callout boxes
3. **Worksheet/exercise** — Guided practice with instructions and self-check
4. **Complete module** — All content types for a single module

Ask the user for:
- Module/lesson topic and learning objectives
- Target audience level
- Content format preference
- Approximate length or duration
- Any specific examples, stories, or case studies to include
- Tone (casual/conversational, professional, academic)

Follow the lesson-writer skill's structure templates. Include engagement hooks, clear transitions, concrete examples, and a strong call to action. For video scripts, target 130-150 words per minute and include all production cues.

Save as `lesson-[module-number]-[topic].md`
