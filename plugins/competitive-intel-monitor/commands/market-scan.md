# /market-scan [industry or topic]

Scan for market trends using the market-trend-analyzer skill.

## Instructions

1. Read the `skills/market-trend-analyzer/SKILL.md` to load the trend analysis framework
2. Clarify scope: Which market/industry? What time horizon? What decision does this inform?
3. Run multi-framework scanning:
   - PEST analysis (Political, Economic, Social, Technological)
   - Porter's Five Forces assessment
   - Competitive landscape dynamics (M&A, funding, entries/exits, positioning shifts)
4. For each trend, assess: Certainty, Timeframe, Magnitude, Opportunity/Threat
5. Generate the Trend Impact Matrix
6. Provide prioritized strategic implications: Must Act Now / Plan Next Quarter / Monitor / Background
7. Deliver the full Market Trend Report
8. Suggest monitoring cadence for ongoing tracking
