# /battlecard [competitor name]

Generate a sales-ready competitive battlecard using the battlecard-builder skill.

## Instructions

1. Read the `skills/battlecard-builder/SKILL.md` to load the battlecard framework
2. If a competitor name was provided, proceed directly to research. If not, ask which competitor.
3. Research the competitor using web search and any connected social media tools
4. Build a complete battlecard following the template in the skill, including:
   - At-a-glance summary
   - Quick dismiss script
   - 3+ trap questions / landmines
   - Top 5 objections with ARP-pattern responses
   - Head-to-head comparison table
   - Pricing intel
   - Win stories and loss patterns
   - Recent moves (last 90 days)
5. Save the battlecard as a markdown file and present to the user
6. Offer to create stage-specific variants (Discovery, Evaluation, Negotiation)
