# /win-loss [deal context]

Run a structured win/loss analysis using the win-loss-analyzer skill.

## Instructions

1. Read the `skills/win-loss-analyzer/SKILL.md` to load the analysis framework
2. If deal context was provided, extract what you can and ask for missing details. If not, interview the user to capture the full deal context using the Step 1 questions from the skill.
3. Analyze the deal using the DECIDE framework:
   - **D**iscovery & Fit
   - **E**valuation Experience
   - **C**hampion & Stakeholders
   - **I**nfluence of Competition
   - **D**ecision Process
   - **E**conomic & Value Case
4. Classify the outcome (win/loss category)
5. Extract actionable intelligence for Sales, Product, Marketing, and CS
6. Generate the full Win/Loss Analysis Report
7. If this is one of multiple deals, offer to do cross-deal pattern analysis
8. Suggest updating relevant battlecards with findings
