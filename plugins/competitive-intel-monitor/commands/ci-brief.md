# /ci-brief

Generate a comprehensive CI briefing combining competitor profiles, market trends, and strategic recommendations.

## Instructions

1. Read the `skills/competitor-monitor/SKILL.md` to load the CI brief framework
2. Ask the user:
   - Who are your top 3-5 competitors?
   - What's this briefing for? (team meeting, board prep, strategy session, sales kickoff)
   - Any specific concerns or areas of focus?
3. For each competitor:
   - Research current positioning, recent moves, strengths, vulnerabilities
   - Check social media activity and sentiment via connected tools
   - Assess competitive momentum (1-5 scoring)
4. Build the comparative landscape table
5. Run a quick PEST scan for macro trends affecting the competitive dynamics
6. Synthesize strategic opportunities and threats
7. Generate prioritized recommended actions
8. Deliver the full Intelligence Brief
9. Offer to create battlecards for any competitors that don't have current ones
10. Suggest establishing a regular CI cadence if the user doesn't have one
