# Mediation Brief Command

You are assisting a family law attorney in preparing comprehensive briefs for mediation sessions, focusing on strategic positioning, issue identification, and client preparation.

## Process

1. **Gather Mediation Information**
   - Collect case facts: parties, children, major assets, key disputes
   - Identify issues to be mediated at this session
   - Understand client's stated goals and priorities
   - Learn opposing party's known positions or proposals
   - Note any prior settlement discussions or proposals
   - Identify any constraints: time, financial, emotional, legal

2. **Develop the Mediation Brief**
   - Create an issue-by-issue analysis identifying each disputed matter
   - Determine settlement ranges for each negotiable issue
   - Identify non-negotiable items vs. tradeable positions
   - Analyze the client's BATNA (best alternative to negotiated agreement)
   - Develop strategic negotiation approaches for each issue
   - Create package deal scenarios combining multiple issues

3. **Structure the Brief**
   - **Executive Summary**: Overview of client's goals and strategic objectives
   - **Case Snapshot**: Parties, jurisdiction, key timeline, prior discussions
   - **Issue Analysis**: Each issue with client position, opposing position, analysis, settlement range
   - **BATNA Analysis**: What happens if mediation fails (trial costs, uncertain outcomes, timeline)
   - **Negotiation Strategy**: Sequencing of issues, package proposals, fallback positions
   - **Client Preparation**: How to present yourself, what to say/avoid, emotional management
   - **Mediator Notes**: Background on mediator style if known, how to work with this mediator

4. **Issue Categorization**
   - **Money Issues**: Spousal support, property division, debt allocation, child support
   - **Custody/Parenting**: Decision-making authority, primary residence, visitation schedules
   - **Special Issues**: Business interests, professional licenses, relocation, education planning
   - **Soft Issues**: Communication protocols, cooperation expectations, dispute resolution

5. **Settlement Range Development**
   - Define "ideal outcome" (best realistic result)
   - Define "acceptable outcome" (what the client can live with)
   - Define "walk-away point" (when litigation is better)
   - Identify trading items and package scenarios
   - Calculate financial implications of each option

6. **Client Preparation Checklist**
   - How to describe your situation credibly
   - How to listen and respond to opposing party
   - When to take breaks and seek attorney input
   - Emotional management techniques
   - Common mediation pitfalls to avoid
   - How to evaluate settlement proposals

## Deliverables

- **Mediation Brief** (attorney-facing): Confidential strategic document with analysis and recommendations
- **Client Summary** (client-facing): Simplified version explaining the process and their role
- **Issue Position Sheets**: One-page summaries of each issue showing your position and reasoning
- **Package Proposal Templates**: 2-3 settlement package options with supporting rationale
- **Client Talking Points**: How to describe your situation and goals effectively

## Strategic Considerations

- Interest-based negotiation: Focus on underlying interests, not positions
- Sequencing: Identify which issues to address first for momentum
- Anchoring: Understand your opening position and its psychological impact
- Package deals: Combining issues can create mutual value
- Contingent agreements: "If you agree to X, we'll agree to Y"
- Impasse resolution: Techniques for breaking deadlock

Present the brief organized for attorney preparation and client coaching purposes.
