Design a comprehensive dashboard specification for $ARGUMENTS (or a general executive dashboard if no argument provided).

Produce a complete dashboard design document including:

**1. Dashboard Overview**
- Title and purpose
- Primary audience and their key questions
- Decision this dashboard enables
- Refresh cadence

**2. KPI Scorecard (Top Row)**
- List 4-6 headline metrics with: metric name, definition, visualization type (number + trend), target/benchmark, alert threshold

**3. Charts & Visualizations (Body)**
For each chart include:
- Chart title
- Chart type and why it's the right choice
- X-axis, Y-axis, color/series
- Key insight it should reveal
- Filters that apply

**4. Filters & Controls**
- Global filters (date range, segment, etc.)
- Default filter values
- Filter interaction behavior

**5. Layout Wireframe**
Describe the layout in text form (grid positions, relative sizes)

**6. Data Requirements**
- Tables/fields needed for each element
- Calculated fields or custom metrics
- Performance considerations

**7. UX & Design Recommendations**
- Color palette guidance
- Mobile vs. desktop considerations
- Accessibility notes
- Interactivity recommendations

Write this as a professional specification ready to hand to a BI developer.
