# Cross-Sell Command

Analyze a client or client segment to identify cross-sell and upsell opportunities.

## Input

Use the argument to specify what to analyze:
- Single client: Name and current coverage
- Client segment: "auto-only clients", "new parents", "business owners", etc.
- Book analysis: "full book analysis", "Q2 book review", etc.

Example: `cross-sell "auto-only clients over $75K income"`

## Analysis Steps

1. **Profile Current Situation**
   - Document all existing policies by line
   - Note policy limits, deductibles, premiums
   - Identify demographic factors: age, family, business, home ownership
   - Record engagement history and prior receptivity

2. **Life Event Scan**
   - Check for major life events in past 6-12 months
   - Use triggers from references/life-events-triggers.md
   - Identify upcoming events (anniversaries, transitions)
   - Note seasonal relevance

3. **Coverage Gap Analysis**
   - Compare existing coverage against comprehensive benchmarks
   - Identify missing lines: auto-only clients need home, home-only need auto, etc.
   - Assess multi-line coordination gaps
   - Note special needs: business, high net worth, specific exposures

4. **Opportunity Prioritization**
   - Score each opportunity using 3-factor model:
     - Fit (1-5): How well does coverage match actual exposure?
     - Receptivity (1-5): How likely to buy based on history?
     - Revenue (1-5): Annual premium potential impact?
   - Calculate combined score (max 15)
   - Rank by score: 12+ High Priority, 9-11 Medium, <9 Lower

5. **Mono-to-Multi Strategy**
   - For auto-only: Lead with bundled auto/home value
   - For home-only: Lead with comprehensive home+auto coordination
   - For business owners: Ensure complete business package
   - Use bundling discount calculations for value prop

6. **Account Rounding Plan**
   - Identify top 20-30 opportunities by priority
   - Suggest outreach timeline (bundled with renewal, separate call, etc.)
   - Recommend approach: email, call, service review meeting
   - Project conversion rates and revenue impact

## Output Format

Provide:
- **Client/Segment Profile** (current coverage, demographics, engagement)
- **Top Opportunities** (ranked by priority score with details)
  - Opportunity type (e.g., "Auto + Home Bundle")
  - Fit/Receptivity/Revenue scores
  - Current gap and recommended coverage
  - Estimated annual premium and bundling savings
- **Recommended Approach** (email, call, service review)
- **Messaging Theme** (value prop for this specific opportunity)
- **Implementation Timeline** (when to approach, follow-up cadence)
- **Projected Outcomes** (conversion likelihood, estimated annual revenue)

## Success Metrics

- Opportunity identification accuracy (matches client needs)
- Recommended approach receptivity (measured by response rate)
- Cross-sell conversion rate (% who adopt recommendations)
- Average revenue per opportunity
- Bundling discount capture rate
