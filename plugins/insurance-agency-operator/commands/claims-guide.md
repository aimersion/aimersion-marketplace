# Claims Guide Command

Generate comprehensive claims filing guidance for a specific claim type.

## Input

Use the argument to specify the claim type:
- auto (auto accident, theft, vandalism)
- home (property damage, water damage, weather)
- liability (injury, property damage)
- life (death claim)
- disability (disability income)

Example: `claims-guide "auto accident with injury"`

## Guidance Steps

1. **Claim Type Confirmation**
   - Identify specific claim type
   - Note any special circumstances (injury, multiple parties, etc.)
   - Confirm applicable policy type and coverage
   - Identify any potentially excluded scenarios

2. **Immediate Action Steps**
   - Safety-first steps (medical attention, emergency services)
   - Scene documentation requirements
   - Key information to gather
   - Timeline for notification to insurer
   - Do's and don'ts specific to claim type

3. **Documentation Checklist**
   - Use detailed checklist from references/claims-checklist.md
   - Tailor checklist to specific claim type
   - Prioritize critical documents
   - Explain what each document proves/supports
   - Note timeline for gathering each item

4. **Claims Process Walkthrough**
   - Step-by-step process from notification through resolution
   - What happens at each phase
   - Timeline expectations
   - Who to contact and when
   - What to expect from insurer

5. **Status Update Framework**
   - How to track claim progress
   - Key checkpoints in process
   - Communication cadence expectations
   - When to follow up if delayed
   - Documentation of all conversations

6. **Advocacy and Problem-Solving**
   - Common issues for this claim type
   - Dispute resolution options if needed
   - When to request appraisal vs. appeal
   - Documentation to support any dispute
   - Escalation path if needed

7. **Claim-Specific Tips**
   - Common mistakes to avoid
   - Red flags to watch for
   - Policyholder responsibilities
   - Coordination with other coverage
   - Timing considerations

## Output Format

Provide:
- **Immediate Actions** (numbered list, prioritized by urgency)
  - Safety first
  - Documentation steps
  - Notification requirements
- **Documentation Checklist** (tailored to claim type)
  - Critical documents (required)
  - Supporting documents (helpful)
  - Gathering timeline
- **Claims Process Timeline** (expected phases and duration)
  - Notification phase
  - Investigation phase
  - Evaluation phase
  - Settlement phase
  - Resolution phase
- **Status Tracking Guide** (how to monitor progress)
  - Key contacts and phone numbers
  - Documentation requests to expect
  - Inspection/appraisal processes
  - Common delays and how to address
- **Advocacy Strategies** (if claim is denied or underpaid)
  - Appeal process
  - Appraisal rights
  - Documentation for dispute
  - Escalation options
- **Claim-Type-Specific Tips** (dos and don'ts)
  - What hurts or helps this type of claim
  - Language to use/avoid
  - Coordination with other parties
  - Long-term claim impacts

## Success Metrics

- Policyholder confidence in claims process
- Documentation completeness (% of required docs submitted)
- Claims settlement timeliness
- Claim approval rate (not denied)
- Client satisfaction with claims guidance
