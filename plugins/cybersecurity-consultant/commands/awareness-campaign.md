Design a comprehensive security awareness training campaign.

Gather:
- Organization size and industry
- Current security awareness maturity
- Previous training results (if any)
- Phishing simulation history
- Specific concerns (phishing, social engineering, data handling, remote work)
- Training delivery preferences (in-person, video, interactive, email)
- Budget and timeline

Produce a security awareness campaign plan:

## 1. Campaign Overview
- Campaign name and theme
- Objectives and success metrics
- Duration and cadence
- Target audience segments

## 2. Training Curriculum
### Module 1: Phishing and Social Engineering
- Recognition techniques
- Reporting procedures
- Real-world examples (sanitized)

### Module 2: Password and Authentication
- Strong password creation
- Password manager usage
- MFA importance and setup

### Module 3: Data Handling and Classification
- Data classification levels
- Handling procedures by classification
- Clean desk policy
- Secure file sharing

### Module 4: Remote Work Security
- VPN usage
- Home network security
- Physical security for devices
- Public Wi-Fi risks

### Module 5: Incident Reporting
- What to report
- How to report
- When to report
- No-blame culture messaging

## 3. Phishing Simulation Plan
- Simulation schedule (monthly recommended)
- Template progression (easy → moderate → advanced)
- Reporting and metrics
- Remedial training triggers

## 4. Communication Materials
- Campaign launch announcement
- Monthly security tips (12 months)
- Poster/digital signage content
- Executive messaging to endorse the program

## 5. Measurement Framework
- Training completion rates
- Phishing simulation click rates (target: <5%)
- Incident reporting rates
- Knowledge assessment scores
- Year-over-year improvement targets

Save all materials as downloadable documents.
