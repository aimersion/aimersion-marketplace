description: Draft a service completion report for a customer
allowed-tools: Read, Write

Draft a professional service completion report to send to a pool service customer.

Ask the user for:

1. **Customer first name**
2. **Date of service**
3. **Technician name**
4. **Services performed** (e.g., chemical check, filter clean, equipment repair)
5. **Water chemistry readings** (before and/or after service)
6. **Any issues found or observations**
7. **Recommendations** (repairs, upgrades, or follow-up actions)
8. **Next scheduled service date**
9. **Company name** (or use the business name if previously established)

Use the customer-communicator skill to draft the report.

Follow these guidelines:
- Keep it friendly and professional — first-name basis with the customer
- Use plain language — no technical jargon or abbreviations the homeowner wouldn't understand
- Translate chemistry readings into simple terms (e.g., "All levels are in the ideal range" or "We adjusted your pH which was slightly high")
- If there are recommendations, explain the benefit to the homeowner, not just the technical issue
- End with the next service date and an invitation to reach out with questions
- Keep it concise — aim for a message that takes 30 seconds to read

Present the draft report ready to send:

```
Subject: Pool Service Complete — [Date]

Hi [Name],

[Body of report]

[Company Name]
```

After presenting the draft, ask if the user wants to adjust the tone, add/remove details, or if it's ready to send.
