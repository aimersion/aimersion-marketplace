# Quarterly Letter Command

Write a compelling quarterly client letter that explains portfolio performance, market context, and strategic positioning.

## Purpose

Create client communications that build confidence, explain investment results, and reinforce long-term strategy through clear, empathetic language.

## Inputs

The user should provide:
- **Quarter and Year**: Q1/Q2/Q3/Q4 and year (e.g., Q4 2025)
- **Client/Account Identifier**: For personalization
- **Portfolio Returns**: Gross and net returns for period
- **Benchmark Performance**: Comparison index performance
- **Market Context**: Key events and economic data
- **Portfolio Changes**: Rebalancing or significant trades
- **Market Environment**: Conditions clients experienced
- **Key Metrics**: Returns, risk metrics, important changes
- **Tone**: Conservative/growth, risk-focused, opportunity-focused

## Usage Pattern

```
Write a quarterly letter for Q1 2026.
Client: [NAME]
Portfolio return: [X]% (benchmark: [Y]%)
Market context: [KEY EVENTS]
Key message: [PRIMARY POINT TO MAKE]
Customization: [SPECIAL CIRCUMSTANCES]
```

## Output Includes

1. **Opening Section**
   - Warm, personal greeting
   - Period summary (quarter and key numbers)
   - One key takeaway
   - Appropriate market context reference

2. **Market Context Section**
   - Summary of major market movements
   - Asset class performance overview
   - Key economic developments
   - Explanation of what drove markets
   - Connection to client's portfolio

3. **Portfolio Performance Section**
   - Period return (absolute percentage)
   - Comparison to benchmark
   - Gross and net-of-fees breakdown
   - Year-to-date and longer-term context
   - Attribution narrative (what drove the return)
   - Performance consistency message

4. **Strategic Positioning Section**
   - How diversification helped/will help
   - Why the portfolio performed as it did
   - Risk management elements at work
   - Long-term strategy alignment
   - Confidence reinforcement

5. **Actions and Updates Section**
   - Rebalancing or allocation changes
   - Upcoming items to discuss
   - Questions answered
   - Planning items to address
   - Invitation to next meeting

6. **Closing Section**
   - Gratitude for trust and partnership
   - Contact information
   - Signature and date

## Letter Templates by Situation

### Strong Performance Quarter
- Celebrate gains without overconfidence
- Reinforce strategy as enabler
- Highlight how diversification helped
- Maintain perspective on volatility

### Weak Performance Quarter
- Acknowledge concerns directly
- Provide historical context (recoveries)
- Explain diversification benefit
- Reinforce discipline and perspective
- Show how portfolio was positioned for protection

### Mixed/Neutral Performance
- Explain differential asset class performance
- Show selective exposure benefit
- Highlight rebalancing opportunities
- Address longer-term positioning

### Market Volatility Quarter
- Name the volatility directly
- Provide historical comparison
- Explain portfolio positioning
- Reinforce staying invested
- Address concerns naturally

## Tone and Language Guidelines

**Strong Opening**
```
Dear [Client(s)],

As [QUARTER] comes to a close, we wanted to share an update on your
portfolio's performance and discuss the market environment during
this [CHARACTERIZATION] period.
```

**Market Context Example**
```
Markets experienced [KEY EVENT] in [QUARTER], driven largely by
[MACRO FACTOR]. The [ASSET CLASS] asset class particularly benefited
from [EXPLANATION], while [ASSET CLASS] faced headwinds due to [REASON].
```

**Performance Attribution Example**
```
Your portfolio returned [X]% during the quarter, compared to
[BENCHMARK]'s [Y]%. This outperformance reflects our positioning in
[SPECIFIC HOLDINGS], which [EXPLANATION]. At the same time, your
[DIVERSIFYING HOLDINGS] provided [BENEFIT].
```

**Strategic Reinforcement Example**
```
This performance demonstrates why we built your portfolio with
[KEY STRATEGIC ELEMENT]. By [STRATEGY], we were able to capture
gains in strong markets while protecting against [RISK]. This is
exactly how we intended your portfolio to work in this environment.
```

## Customization by Client Type

### Conservative Income-Focused Clients
- Emphasize capital preservation
- Highlight income generation
- Address volatility in growth portion
- Show cushion from bonds/stable value
- Discuss inflation/purchasing power risk

### Growth-Focused Clients
- Celebrate gains (without hubris)
- Show long-term compounding benefit
- Emphasize staying invested through volatility
- Highlight opportunity in downturns
- Reinforce time horizon

### High-Net-Worth Clients
- Deeper performance attribution
- Alternative asset performance
- Tax implications (if relevant)
- Concentrated position management
- Estate/charitable planning updates

### Newly Retired Clients
- Income coverage emphasis
- Withdrawal sustainability
- Sequence of returns management
- Spending discipline reinforcement
- Longevity planning

## Key Elements to Include

**Always Include:**
- Period return (gross and net)
- Benchmark comparison
- Key market events
- Performance explanation
- Long-term perspective
- Gratitude and next steps

**Vary Based on Situation:**
- Market volatility messaging
- Risk management accomplishments
- Rebalancing rationale
- Tax management results
- Upcoming planning items

## Formatting Recommendations

- **Length**: 1-2 pages (keep concise)
- **Readability**: Short paragraphs, clear language
- **Charts**: Include if they aid understanding
- **Tables**: Use for performance data
- **Tone**: Warm and professional
- **Fonts**: Conservative and readable
- **Signature**: Hand-signed when possible

## Distribution

- Email with PDF attachment
- Printed and mailed (more formal)
- Combined with performance statement
- Within X days of quarter end
- Archive in compliance file

## Integration with Other Skills

- **Portfolio Reporter**: Use performance analysis as source
- **Client Communicator**: Leverage broader communication frameworks
- **Compliance Tracker**: Document that letters sent and received
- **Tax Strategist**: Include tax results if year-end letter

## Notes

- Personalize for client situation and preferences
- Avoid financial jargon; explain concepts simply
- Be honest about performance (good and bad)
- Provide context (don't minimize or exaggerate)
- Reinforce investment discipline
- Invite questions and conversation
- Archive copies for compliance
- Consider companion quarterly call/meeting
