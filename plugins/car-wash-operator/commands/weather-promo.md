Create a weather-triggered marketing campaign for the car wash.

Ask the user for:
- Current or upcoming weather condition (rain, pollen, snow/salt, heat wave, post-storm, etc.)
- Target audience (all customers, members only, new customers)
- Preferred channels (text, social media, email, in-store signage)
- Any existing promotions to coordinate with

Generate a complete weather campaign using the weather-marketing skill:

1. **Campaign message**: Ready-to-send texts, social media posts, and/or email copy tailored to the weather event
2. **Timing**: When to send relative to the weather event for maximum impact
3. **Offer** (if appropriate): Specific discount or incentive tied to the weather condition
4. **Rain guarantee messaging** (if applicable): Reinforce the rain guarantee to overcome weather objections
5. **Multiple channels**: Provide formatted content for each channel the user wants to use

Make all content ready to copy-paste and send. Include relevant weather-themed language and urgency drivers.
