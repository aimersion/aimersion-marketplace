Generate an electrical job estimate for: $ARGUMENTS

1. Read the job-estimator skill and its reference pricing data
2. Ask clarifying questions if needed: job type (service/residential/commercial), specific scope, location
3. Calculate material costs using reference pricing, adjusted for current market
4. Calculate labor hours using benchmark data, assign journeyman vs. apprentice hours
5. Apply billing multiplier to cover burden, overhead, and profit
6. Add permit fees, material tax, and any equipment rental
7. Present a clear estimate with line-item detail and a total
8. Offer to generate a professional estimate document for the customer
