Create apprentice training content for: $ARGUMENTS

1. Read the apprentice-trainer skill for curriculum structure and NEC references
2. Determine what is needed:
   - If an apprentice year is specified: generate a quarterly training plan for that year
   - If a specific topic: create a detailed training task assignment
   - If "exam prep": generate a journeyman exam study plan
3. Include NEC code references for every skill area
4. Specify safety considerations for each task
5. Define clear competency criteria (what "done right" looks like)
6. Format as a structured document the user can print or share with their apprentice
