Optimize collection page merchandising, sorting, filtering, and cross-sell strategies.

## Steps

1. **Gather info** — Ask for:
   - Collection name and URL
   - Number of products in the collection
   - Current sort order
   - Product category
   - Top-selling products in this collection
   - Any current cross-sell or upsell setup

2. **Build strategy** — Following the collection-merchandiser skill:
   - Recommended sort strategy
   - Top 8 grid positions with reasoning
   - Filter recommendations
   - Cross-sell pairings
   - Badge strategy
   - Collection page SEO (title, meta, description)

3. **Deliver** — Present in the merchandising deliverable format
