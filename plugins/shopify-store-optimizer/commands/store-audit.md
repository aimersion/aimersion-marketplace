Run a comprehensive audit of a Shopify store. Ask the user for their store URL and any specific concerns.

## Steps

1. **Gather info** — Ask for:
   - Store URL
   - Current conversion rate (if known)
   - Monthly traffic estimate
   - Product category / industry
   - Biggest concern or goal

2. **Speed check** — Search for the store's PageSpeed Insights scores and Core Web Vitals

3. **Visual audit** — If browser tools are available, visit the store and evaluate:
   - Homepage clarity and value proposition
   - Product page quality (copy, images, trust signals)
   - Mobile experience
   - Navigation and search
   - Checkout options visible

4. **SEO check** — Look at title tags, meta descriptions, schema markup

5. **Score and prioritize** — Rate each area 1-5 and generate a prioritized action plan

6. **Deliver report** — Save as a markdown file with the scoring template from the store-auditor skill

## Output
Present findings in the scoring template format from the store-auditor SKILL.md with clear, numbered priority fixes.
