Write conversion-optimized product page copy using proven copywriting formulas.

## Steps

1. **Gather product info** — Ask for:
   - Product name and category
   - Key features and specifications
   - Target customer (who buys this)
   - Price point
   - Primary keyword for SEO
   - Existing description (if rewriting)
   - What makes this product different from alternatives

2. **Select formula** — Based on product type:
   - Default: PAS + FAB hybrid
   - High-ticket ($200+): 4Ps
   - Lifestyle brand: Star Story Solution
   - Technical product: FAB-led
   - Problem-solving product: PAS-led

3. **Write copy** — Generate the full product page copy following the product-page-optimizer skill structure:
   - Title tag (50-60 chars)
   - Meta description (150-160 chars)
   - H1 headline (benefit-focused)
   - Product description (80-120 words)
   - Trust signal recommendations
   - CTA text

4. **Deliver** — Present with SEO notes, pricing psychology tips, and mobile optimization reminders
