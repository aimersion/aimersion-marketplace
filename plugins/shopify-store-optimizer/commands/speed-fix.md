Diagnose speed issues and create a prioritized fix plan.

## Steps

1. **Get the URL** — Ask for the store URL

2. **Measure** — Search for PageSpeed Insights data or ask the user to share their scores:
   - Mobile and desktop scores
   - LCP, CLS, INP values
   - Top issues flagged

3. **Diagnose** — Check the 6 speed killers in order:
   - Unoptimized images
   - App bloat
   - Theme issues
   - Third-party scripts
   - Render-blocking resources
   - Shopify-specific issues

4. **Build fix plan** — Following the speed-optimizer skill:
   - Prioritized list of fixes
   - Quick wins (under 1 hour)
   - Medium-term improvements
   - Estimated impact of each fix

5. **Deliver** — Present in the speed audit report format
