Draft a professional guest communication of type: $ARGUMENTS

Follow these steps:

1. **Identify message type** — Determine which guest journey stage this message is for (pre-arrival, in-stay welcome, mid-stay check-in, service recovery, post-stay thank you, review response, re-engagement). If not specified in the arguments, ask.

2. **Gather details** — Ask for the essential context:
   - Guest name and stay details (dates, room type)
   - Property name and positioning (luxury, boutique, business, resort)
   - For service recovery: what went wrong and what was done to fix it
   - For review responses: the review text and platform
   - For re-engagement: time since last stay and any offers to include

3. **Draft the message** — Write the communication following the guest-communicator skill principles: warm and personal, anticipatory, solution-oriented, brand-consistent, and action-clear.

4. **Provide variants** — Offer 2 versions: one formal/polished and one warm/conversational, so the user can choose the tone that matches their brand.

Present both versions clearly labeled, ready to copy and send.
