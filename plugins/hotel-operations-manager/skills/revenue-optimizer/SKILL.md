# Hotel Revenue Optimization

Drive RevPAR growth through data-informed rate strategy, demand pattern analysis, and competitive positioning.

## Core Framework

### Key Performance Metrics

Track and optimize around these industry-standard KPIs:

- **ADR** (Average Daily Rate) — total room revenue / rooms sold
- **Occupancy Rate** — rooms sold / rooms available
- **RevPAR** (Revenue Per Available Room) — ADR x occupancy rate
- **GOPPAR** (Gross Operating Profit Per Available Room) — total GOP / available rooms
- **TRevPAR** (Total Revenue Per Available Room) — includes F&B, spa, ancillary
- **Rate Index** — property ADR / comp set ADR (target >1.0)

### Rate Strategy Development

When building a rate strategy:

1. **Analyze demand calendar** — identify compression nights (citywide events, holidays, peak seasons), shoulder periods, and need dates
2. **Map room type mix** — evaluate rate tiers across standard, premium, suite categories; identify upgrade path revenue opportunities
3. **Set rate fences** — define restrictions by advance purchase window, length of stay, cancellation policy, package inclusions
4. **Channel strategy** — allocate inventory across direct website, OTA (Booking.com, Expedia), GDS, wholesale, and opaque channels with appropriate rate differentiation
5. **Dynamic pricing rules** — establish floors, ceilings, and automated adjustment triggers based on pace-to-budget, booking velocity, and pickup patterns

### Demand Forecasting

Build forecasts using:

- **Historical patterns** — same-period prior year, day-of-week trends, seasonal curves
- **Pace analysis** — current bookings on-the-books vs. same-time-last-year pickup
- **Market intelligence** — event calendars, airline seat capacity, destination marketing campaigns
- **Segmentation** — forecast by segment (transient, corporate negotiated, group, wholesale) separately then aggregate

### Competitive Set (Comp Set) Analysis

Evaluate positioning against 4-6 comparable properties:

- Match by location proximity, star class, room count, amenity level, and target market
- Track rate shopping data across major OTAs weekly
- Calculate Rate Index and Occupancy Index to identify whether you're winning on price, volume, or both
- Identify displacement opportunities when comp set sells out

### Revenue Action Plan Template

Structure recommendations as:

1. **Current performance snapshot** — MTD and YTD vs. budget and prior year
2. **30/60/90-day demand outlook** — expected compression, soft periods, events
3. **Rate recommendations by date range** — specific BAR adjustments, restrictions, promotions
4. **Channel optimization** — where to push/pull inventory
5. **Upsell and ancillary opportunities** — packages, upgrades, F&B bundles
