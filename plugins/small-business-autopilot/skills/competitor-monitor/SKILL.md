# Competitor Monitor

Research competitors and synthesize actionable competitive intelligence for small business owners.

## Core Workflow

1. **Identify competitors** — confirm the list of competitors to research
2. **Research each competitor** — web presence, social media, messaging, offers
3. **Analyze patterns** — positioning, pricing signals, content strategy, strengths/weaknesses
4. **Deliver the brief** — structured report with actionable recommendations

## Research Methodology

For each competitor, investigate:

### Online Presence
- Website messaging: headline, value proposition, key differentiators
- Pricing (if publicly available): tiers, positioning, comparison to user's pricing
- Recent changes: new products, redesigns, pivots

### Social Media Activity
- Search X for recent posts from/about the competitor
- Check their LinkedIn company page for recent content themes
- Review Facebook page for engagement patterns and customer sentiment

### Content & SEO
- Blog topics and publishing frequency
- Lead magnets and content offers
- Search visibility for key industry terms

### Customer Sentiment
- Look for reviews, testimonials, and public feedback
- Note common praise and complaints
- Identify gaps competitors aren't addressing

## Competitive Brief Structure

Present findings in this format:

```
# Competitive Intelligence Brief
Date: [date]
Prepared for: [user's business]

## Competitor Overview

### [Competitor 1]
**Positioning**: [one sentence]
**Key differentiators**: [bullets]
**Recent activity**: [notable moves in last 30 days]
**Strengths**: [what they do well]
**Weaknesses**: [where they fall short]

### [Competitor 2]
[same structure]

## Comparative Analysis

| Dimension | Your Business | Competitor 1 | Competitor 2 |
|-----------|--------------|-------------|-------------|
| Positioning | | | |
| Pricing | | | |
| Key strength | | | |
| Content frequency | | | |

## Opportunities
[2–3 actionable insights the user can act on]

## Threats to Watch
[1–2 competitive moves that could impact the user's business]

## Recommended Actions
[3–5 specific, prioritized actions]
```

## Research Tools

Use web search to find current information about competitors. Search for:
- "[Competitor name] + recent news"
- "[Competitor name] + reviews"
- "[Competitor name] + pricing"
- Industry-specific terms + competitor names

Use connected ~~social media tools to search for competitor mentions and activity on X, LinkedIn, and Facebook.

## Weekly Monitoring

For ongoing monitoring, track these signals weekly:
- New blog posts or content published
- Social media posting frequency and engagement changes
- Pricing or offer changes
- New product/service launches
- Customer reviews and sentiment shifts
- Job postings (indicate growth areas or new initiatives)
