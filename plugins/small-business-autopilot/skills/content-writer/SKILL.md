# Content Writer

Create high-quality marketing content optimized for search engines and conversions.

## Core Workflow

1. **Understand the brief** — topic, target audience, goal, target keywords
2. **Research** — search for current information, competitor content, and keyword context
3. **Outline** — propose a structure for approval before writing
4. **Draft** — write the full piece following SEO and copywriting best practices
5. **Deliver** — present the finished content with metadata

## Blog Post Writing

### SEO Blog Post Structure

```
Title: [Primary keyword near the front, compelling, 50–60 characters]
Meta description: [150–160 characters, includes primary keyword, has a CTA]

## Introduction (100–150 words)
- Open with the reader's problem or question
- Establish why this matters NOW
- Preview what they'll learn
- Include primary keyword naturally in first 100 words

## [H2: Subtopic with Secondary Keyword]
- 200–400 words per section
- Use H3s to break up long sections
- Include examples, data, or actionable advice

## [H2: Next Subtopic]
[Continue pattern]

## [H2: Actionable Takeaway or Summary]
- Recap key points
- Give a clear next step

## Call to Action
- Link to relevant product/service
- Invite comments or questions
- Suggest related content
```

### SEO Writing Rules
- Primary keyword in: title, first paragraph, one H2, meta description, and naturally 3–5 more times
- Secondary keywords sprinkled in H2s and body text
- Internal links: 2–3 links to other relevant pages on the user's site
- External links: 1–2 links to authoritative sources
- Ideal length: 1,200–2,000 words for most topics (longer for competitive keywords)
- Use short paragraphs (2–4 sentences), bullet points, and subheadings for readability
- Write at an 8th-grade reading level for maximum accessibility

## Landing Page Copy

### Structure
1. **Hero section**: Headline (benefit-driven) + subheadline (supporting detail) + primary CTA
2. **Problem**: Articulate the pain point the reader is experiencing
3. **Solution**: Introduce the product/service as the answer
4. **Benefits**: 3–5 key benefits (not features) with supporting details
5. **Social proof**: Testimonials, stats, logos, case studies
6. **FAQ**: Address top 3–5 objections
7. **Final CTA**: Repeat the offer with urgency if appropriate

### Copywriting Principles
- Lead with benefits, support with features
- Use "you" and "your" — make it about the reader
- One CTA per page (can repeat it, but don't offer competing actions)
- Use specific numbers over vague claims ("97% of customers" not "most customers")
- Address objections directly rather than ignoring them

## Other Content Types

### Product/Service Descriptions
- Lead with the transformation or outcome
- Follow with how it works (3 simple steps)
- Include specifications or details
- End with CTA

### About Page
- Open with the customer's story, not yours
- Transition to your origin story and mission
- Introduce the team with personality
- End with values and a CTA

### Google Business Profile Posts
- 150–300 words
- Include a relevant keyword
- Add a CTA with link
- Post weekly for local SEO benefit

## Deliverable Format

Present each piece with:
```
Content type: [Blog post / Landing page / etc.]
Target keyword: [primary keyword]
Secondary keywords: [2-3 secondary keywords]
Word count: [n]
Meta title: [50-60 chars]
Meta description: [150-160 chars]

[Full content]
```
