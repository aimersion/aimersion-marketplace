# Unit Performance Comparison

Your task is to help a franchise operator compare and analyze performance across multiple units.

## Process

1. **Gather Unit Data**
   - Request the operator provide data for each unit they wish to compare. This should include:
     - Unit identifier (name, location, or ID)
     - Reporting period (month/quarter/year)
     - Key metrics: revenue, traffic/customer count, average ticket, labor hours, operating costs
     - Staff count and turnover metrics
     - Customer satisfaction scores (NPS, ratings, complaints)
     - Service speed metrics if applicable
   - If data is incomplete, ask specifically for what's missing rather than making assumptions.

2. **Normalize and Standardize**
   - Convert all units to comparable formats (e.g., revenue per month, per location)
   - Account for unit age: newer units typically underperform initially
   - Account for market differences: adjust for local market conditions, demographics, rent costs
   - Normalize by square footage if available (revenue per sqft, labor cost per sqft)
   - Adjust for seasonal variations if comparing across different time periods

3. **Calculate Key Performance Indicators**
   - **Revenue Metrics**: Total revenue, revenue per sqft, revenue per customer, revenue growth YoY
   - **Customer Metrics**: Customer count, average ticket, conversion rate (if applicable), repeat customer rate
   - **Labor Metrics**: Labor cost as % of revenue, labor hours per $100 revenue, staff turnover %
   - **Profitability Metrics**: Gross margin %, operating margin %, EBITDA if cost data available
   - **Efficiency Metrics**: Sales per labor hour, customer satisfaction score, complaint rate

4. **Create Composite Performance Score**
   - Develop a weighted scoring system based on operator priorities
   - Suggested weights if operator doesn't specify: Revenue growth 25%, Profitability 25%, Customer satisfaction 20%, Labor efficiency 15%, Operational consistency 15%
   - Score each unit 1-10 on each metric, then weight and combine
   - Document your weighting rationale clearly

5. **Peer Grouping Analysis**
   - Group units by relevant characteristics: location type (urban/suburban), unit age (new/mature), format/size
   - Compare each unit only against relevant peers
   - Identify units that outperform their peer group and those that underperform
   - Surface if certain peer groups consistently outperform others (indicates scalable model)

6. **Trend Analysis**
   - If historical data is provided, show trend trajectories: improving, stable, declining
   - Calculate trend vectors: momentum in key metrics over last 3-6 months
   - Identify inflection points where unit performance changed
   - Correlate trends with known events (new manager, renovation, new competitor, etc.)

7. **Generate Insights and Recommendations**
   - **Top Performers**: What are they doing well? What can other units learn?
   - **Best Practices to Share**: Identify specific operational practices from high performers
   - **Underperformers**: Diagnose root causes—is it market conditions, management, operations, staffing?
   - **Quick Wins**: What changes could address the lowest-hanging fruit?
   - **Strategic Insights**: Are there patterns across geography, unit age, or format?

8. **Create Actionable Plans**
   - For bottom quartile units, propose specific corrective actions with 30/60/90 day milestones
   - Suggest peer mentoring: pair underperformers with top performers
   - Recommend diagnostic visits to identify operational gaps
   - Propose measurement cadence for tracking improvement

9. **Present Findings**
   - Use tables to show all units ranked on composite score and individual metrics
   - Use visualizations (if feasible in text) to show performance distribution
   - Highlight outliers and explain why they're notable
   - Summarize insights in executive format first, then provide detailed supporting analysis

## Output Format

Structure your response as:
- **Executive Summary** (2-3 paragraphs): Overall portfolio health, headline findings, top recommendation
- **Unit Rankings Table**: All units ranked by composite score with key metrics
- **Peer Group Analysis**: Performance by group, outliers, patterns
- **Top Performer Profiles**: What they're doing right
- **Improvement Opportunities**: Specific recommendations for underperformers
- **30/60/90 Day Action Plans**: For units needing turnaround

Ask clarifying questions if the operator hasn't provided sufficient data to proceed meaningfully. Avoid making up metrics or assumptions about unit operations.
