Write a complete SEO-optimized blog post targeting the keyword: $ARGUMENTS

Follow the blog-writer skill methodology. Complete these steps:

1. **Create a content brief first:**
   - Search the web for the top 5 pages currently ranking for this keyword
   - Document the winning content format, average word count, topics covered, and gaps
   - Identify SERP features (featured snippets, PAA questions, AI Overview presence)
   - Define the content angle — what will make this piece better than what exists
   - Identify 3-5 secondary keywords and 2-3 PAA questions to answer

2. **Plan the post structure:**
   - Determine the best format (how-to, listicle, guide, comparison) based on SERP analysis
   - Outline H2/H3 structure with notes on what each section covers
   - Map keyword placement across sections
   - Plan featured snippet optimization (which question, which format)

3. **Write the SEO metadata:**
   - Meta title (50-60 chars, keyword in first 5 words, compelling)
   - Meta description (150-160 chars, unique value prop + CTA)
   - URL slug (3-5 words, keyword-rich, no dates)

4. **Write the full article:**
   - Hook introduction (no "In today's world..." — use a stat, pain point, or scenario)
   - Primary keyword in first 100 words
   - Well-structured body with H2/H3 hierarchy
   - For each major section: specific examples, data, or original insights
   - Short paragraphs (2-4 sentences), active voice, Flesch 60+
   - Featured snippet targets: 40-60 word answer blocks after relevant headings
   - FAQ section answering PAA questions with concise answers
   - Strong conclusion with clear CTA

5. **Apply E-E-A-T signals:**
   - Include author byline placeholder
   - Cite 2-4 authoritative external sources
   - Include specific examples, numbers, or case studies
   - Add "Last updated: [date]" metadata note

6. **Generate schema markup:**
   - Article/BlogPosting JSON-LD with author, dates, publisher
   - FAQ JSON-LD if the post answers questions
   - Provide ready-to-implement code blocks

7. **Complete the SEO checklist:**
   - Keyword placement verified (title, H1, first 100 words, H2, description, slug)
   - 3-5 internal links with descriptive anchor text
   - 2-4 external links to authoritative sources
   - Image placement suggestions with alt text
   - Mobile-friendly formatting confirmed

Save the completed post as a markdown file with: content brief, SEO metadata, full article, schema markup code, internal link suggestions, image notes, and featured snippet targets.
