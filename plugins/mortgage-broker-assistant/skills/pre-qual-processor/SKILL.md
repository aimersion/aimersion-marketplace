# Pre-Qual Processor Skill

## Overview
Streamline pre-qualification and pre-approval processes by maintaining accurate document checklists for each loan product, generating professional pre-qual letters, addressing common documentation issues, and providing income verification guidance.

## Core Responsibilities

### Loan-Type-Specific Document Checklists
Maintain complete document requirements for:
- **Conventional Loans** (Fannie Mae/Freddie Mac guidelines)
- **FHA Loans** (HUD guidelines and lender requirements)
- **VA Loans** (VA guidelines and lender-specific requirements)
- **USDA Loans** (USDA RHS guidelines)
- **Jumbo Loans** (portfolio and investor standards)
- **Non-QM Loans** (alternative qualification products)
- **Self-Employed/1099 Income** (dual tax return analysis)

Each checklist includes:
- Required documents with filing requirements
- Optional documents for condition clearance
- Timing windows (age of documents)
- Format specifications (certified, notarized, etc.)
- Alternative documentation scenarios

### Pre-Qualification Letter Templates
Generate professional pre-qual letters including:
- Borrower name and contact information
- Property address and loan amount
- Pre-qualified rate and estimated monthly payment
- Clear statement of pre-qualification limitations
- Contingencies (appraisal, employment verification, final underwriting)
- Expiration date and document validity period
- Logo, broker information, NMLS licensing
- Compliance language and disclosures

Customize templates for:
- Single borrower vs. co-borrower scenarios
- Conditional pre-qualification language
- High-balance or jumbo product notation
- Construction or non-standard property types

### Common Document Issues and Solutions
Address typical problems including:
- **Stale Documents**: Guidelines on acceptable age thresholds
- **Income Verification**: Multiple acceptable proof forms
- **Employment Gaps**: Explanation letters and verification strategies
- **Credit Issues**: Compensating factors documentation
- **Debt Obligations**: Disclosure requirements and calculation methods
- **Self-Employment**: Dual tax return interpretation
- **Recent Changes**: New employment, bankruptcy, foreclosure disclosures
- **Foreign Income**: ITIN documentation and currency conversion
- **Gift Funds**: Seasoning requirements and donor documentation

Solutions include:
- Acceptable alternatives when primary document unavailable
- Explanation letter guidance and templates
- Compensating factors documentation
- Rehabilitation period timelines

### Income Calculation Guides
Provide calculation methodology for:
- **W-2 Employment**: Gross income interpretation
- **Commission Income**: 2-year average calculation
- **Bonus Income**: Qualification criteria and verification
- **Overtime Income**: Stability analysis and qualification rules
- **Self-Employment Income**: Tax return analysis methodology
- **Rental Income**: Deductible expense treatment
- **Alimony/Child Support**: Qualification requirements
- **Social Security/Pension**: Fixed income documentation
- **Investment Income**: Account statement averaging

## Workflow

1. **Collect Borrower Information**: Gather basic financial and employment details
2. **Identify Loan Product**: Determine program type and pricing
3. **Select Document Checklist**: Provide appropriate checklist for loan type
4. **Identify Gaps**: Highlight missing or deficient documents
5. **Propose Solutions**: Suggest acceptable alternatives and evidence
6. **Calculate Income**: Determine qualifying income from documentation
7. **Generate Pre-Qual Letter**: Create letter with contingencies noted
8. **Establish Timeline**: Set document collection and review deadlines

## Key Outputs

- Customized document checklist (with priority items flagged)
- Pre-qualification letter (PDF-ready format)
- Income calculation summary
- Common issue solutions list
- Document collection timeline
- Pre-approval conditions checklist
- Borrower education materials

## Best Practices

- Provide checklists immediately during initial borrower contact
- Organize documents by category for lender submission
- Request documents in electronic format when possible
- Review documents for quality and legibility before submission
- Maintain document tracking spreadsheet for compliance
- Flag time-sensitive documents (employment verification, account statements)
- Request 2-3 extra copies of each document for underwriting
- Keep pre-qual letters updated if rates or conditions change
- Document explanations for any unusual situations
- Verify employment and bank accounts per lender guidelines
- Maintain calendar of document expiration dates
- Educate borrowers on why each document is required
