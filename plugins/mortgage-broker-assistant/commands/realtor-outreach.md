# Realtor Outreach Command

Create targeted outreach materials and campaigns to build realtor relationships.

## Instructions

Based on the realtor or campaign type in $ARGUMENTS:

1. **Identify campaign type**:
   - Cold outreach (new agent prospecting)
   - Nurture (existing relationship deepening)
   - Event promotion (inviting agent to event)
   - Re-engagement (lapsed referral partner)
   - Extract from $1 or ask user to specify

2. **If cold outreach** ($1 = "cold" or specific realtor name):
   - Create 3-email cold outreach sequence
   - Include: Introduction, follow-up, value proposition
   - Use agent-outreach-email-sequences.md as reference
   - Customize with specific market/neighborhood details
   - Include personalization hooks and call-to-action

3. **If nurture/ongoing** ($1 = "nurture" or "monthly"):
   - Create monthly market update template
   - Create product/guideline update email
   - Create success story/testimonial email
   - Structure for bi-weekly or monthly cadence
   - Include data, lending insights, relevant to agents

4. **If event promotion** ($1 = "event" or specific event):
   - Create event invitation email (with details)
   - Create reminder email (1 week before)
   - Include event date, time, location, agenda
   - Explain value for agents and their clients
   - Call-to-action for RSVP

5. **If re-engagement** ($1 = "re-engagement" or "lapsed"):
   - Create 3-email re-engagement sequence
   - Include: Check-in, value reminder, incentive offer
   - Focus on rebuilding relationship
   - Acknowledge lapse without being pushy

6. **Customize as needed** (if additional details provided):
   - Insert specific realtor name
   - Add company/market-specific details
   - Include actual lending product updates
   - Reference specific market data or deals
   - Add personal details if known

7. **Format as**:
   - Complete email(s) ready to send
   - Subject lines included
   - Personalization placeholders clearly marked
   - Professional tone matching broker brand
   - Clear CTAs and next steps

Use agent-outreach-email-sequences.md and realtor-nurture skill guidance for tone, structure, and content.

## Output

Provide ready-to-use outreach materials:
- Complete email sequence (3+ emails typically)
- Subject lines and customization notes
- Timing recommendations
- Next steps if agent responds/doesn't respond
- Co-marketing material suggestions (if applicable)
