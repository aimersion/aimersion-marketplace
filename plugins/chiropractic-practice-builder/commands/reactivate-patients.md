description: Identify inactive patients and create re-engagement campaigns
allowed-tools: Read, Write, Edit, WebSearch
argument-hint: [segment: cooling-off|lapsed|dormant|all]
---

Use the patient-reactivation skill to help the user win back inactive patients.

1. Ask the user for their patient list or criteria for identifying inactive patients. If they provide a file, read it. If not, help them define segments based on last visit date.

2. Segment patients into tiers: Cooling Off (30–60 days), Lapsed (61–120 days), Dormant (121–365 days), Lost (365+). If $ARGUMENTS specifies a segment, focus on that segment only.

3. For each segment, generate:
   - A personalized text/SMS message template
   - A follow-up email template
   - A phone call script for front desk staff

4. Create a 3-touch outreach sequence with recommended timing between touches.

5. Build a simple tracking spreadsheet template or checklist the user can use to monitor who was contacted and who responded.

6. Save all deliverables to files the user can download and customize.

Keep all messaging warm, personal, and non-pushy. Reference specific chiropractic wellness benefits. Never use fear-based or guilt-based language.
