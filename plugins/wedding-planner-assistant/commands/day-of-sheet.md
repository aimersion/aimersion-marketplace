Create complete day-of coordination documents for: $ARGUMENTS

Follow these steps:

1. **Gather final details** — Ask for: ceremony time, venue(s), vendor list with contact info, wedding party names and roles, processional order, special ceremony elements, reception event sequence, song selections, and any unique logistics.

2. **Ceremony run sheet** — Build a minute-by-minute ceremony document including: pre-ceremony setup checklist, processional order with music cues, ceremony flow, and post-ceremony plan.

3. **Reception run sheet** — Build the reception timeline with: grand entrance, first dance, toasts (speakers and order), dinner service, parent dances, cake cutting, special events, last dance, and send-off.

4. **Vendor contact and tip sheet** — Create the vendor tip envelope list with recommended amounts and distribution plan.

5. **Emergency kit checklist** — Generate the comprehensive emergency kit packing list.

6. **Coordinator checklists** — Produce morning-of, ceremony, and reception execution checklists.

Save all documents to a `day-of-[date]/` folder with separate files for each document.
