Generate a wedding budget report for a total budget of: $ARGUMENTS

Follow these steps:

1. **Establish parameters** — Confirm: total budget, guest count, wedding style (intimate, classic, luxury), location/region, and any fixed costs already committed.

2. **Create budget allocation** — Using the budget-tracker skill framework, build a category-by-category budget with recommended allocations adjusted for their priorities.

3. **Track current spending** — If the couple has already booked vendors, create a budget vs. actual tracking table showing committed, paid, and remaining amounts.

4. **Hidden costs alert** — Flag commonly forgotten expenses relevant to their wedding setup.

5. **Cost-saving opportunities** — Identify 5-10 specific savings strategies based on their wedding details.

6. **Payment schedule** — Create a month-by-month payment calendar for all vendors.

Save to `wedding-budget-report.md`.
