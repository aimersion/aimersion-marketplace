Generate today's crew dispatch assignments.

1. Read the crew-dispatcher skill for dispatch methodology and weather decision matrix
2. Ask the user for: number of crews available, pending jobs with addresses and scope, any priority jobs
3. Check weather forecast for the service area using web search
4. Apply the weather decision matrix — determine if conditions allow roof work
5. Match crews to jobs based on complexity, proximity, and priority
6. Generate a dispatch board showing all crew assignments
7. Create individual crew sheets with job details, materials, and special instructions
8. Flag any jobs that should be rescheduled due to weather or material issues
