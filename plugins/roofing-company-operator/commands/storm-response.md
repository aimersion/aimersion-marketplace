Activate a storm damage response plan for: $ARGUMENTS

1. Read the storm-damage-responder and lead-generator skills
2. If a location is provided, search for recent severe weather reports in that area (hail size, wind speeds, affected neighborhoods)
3. Generate a prioritized action plan covering:
   - Immediate team mobilization steps (first 24 hours)
   - Canvassing route targeting affected neighborhoods
   - Social media posts announcing free inspections
   - Email/text blast to existing customers in the area
   - Door hanger content for the affected area
4. Create a canvassing script tailored to the specific storm event
5. Prepare an inspection report template for documenting damage
6. Output a complete storm response action plan document
