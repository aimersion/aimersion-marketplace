Generate a professional client status update.

Gather:
- Project name and client name
- Sprint number or reporting period
- Work completed this period
- Work planned for next period
- Blockers or risks
- Budget status (hours used vs. planned)
- Any scope change requests

Produce a polished status report:

## 1. Summary
- One-paragraph project health summary
- Overall status: On Track / At Risk / Off Track (with color coding)
- Sprint goal achievement

## 2. Progress Update
- Completed items with brief descriptions
- Demos or deliverables ready for review
- Stories or features moved to done

## 3. Metrics
- Velocity trend (last 3 sprints)
- Budget burn rate vs. plan
- Scope completion % vs. timeline
- Defect count and resolution rate

## 4. Upcoming Sprint Plan
- Planned work items with priorities
- Expected deliverables
- Key milestones approaching

## 5. Risks and Blockers
For each risk/blocker:
- Description
- Impact (High/Medium/Low)
- Mitigation plan
- Owner
- Decision needed from client (if any)

## 6. Action Items
- Table: Action | Owner | Due Date | Status

Keep the tone professional but approachable. Save as a downloadable document.
