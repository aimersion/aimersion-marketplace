Run a comprehensive reputation audit for the user's business.

## Steps

1. **Gather business info** — Ask for:
   - Business name and location
   - Industry
   - Primary review platforms
   - Top 2-3 competitors (optional)

2. **Audit current reputation**:
   - Search the web for the business name + "reviews"
   - Check Facebook page info and recent reviews/comments
   - Search X for brand mentions
   - Search email for recent review notifications
   - Note current ratings on each platform found

3. **Analyze review health** using the review-monitor skill:
   - Calculate the health scorecard metrics
   - Identify the overall grade (A through F)
   - Spot trends and patterns

4. **Competitive comparison** (if competitors provided):
   - Search for competitor reviews on the same platforms
   - Compare ratings, volume, and response quality
   - Identify competitive advantages and gaps

5. **Generate recommendations**:
   - Top 3 immediate actions to improve reputation
   - Long-term reputation strategy (30/60/90 day plan)
   - Review solicitation recommendations
   - Response strategy improvements

## Output Format

```
🔍 REPUTATION AUDIT REPORT
Business: [name]
Date: [date]

## OVERALL GRADE: [A/B/C/D/F]

## PLATFORM RATINGS
[Platform-by-platform summary]

## HEALTH METRICS
[Scorecard from review-monitor]

## SENTIMENT ANALYSIS
[Key themes from reviews]

## COMPETITIVE POSITIONING
[How you compare to competitors]

## RECOMMENDATIONS
🔴 Immediate (This Week): [actions]
🟡 Short-term (30 Days): [actions]
🟢 Long-term (90 Days): [actions]
```
