Generate a comprehensive social media performance report for Aimersion AI across all connected platforms.

## Process

1. **Collect data from connected platforms**:
   - **X**: Retrieve recent posts and engagement metrics using X tools (get timeline, check post metrics)
   - **LinkedIn**: Retrieve recent posts and engagement using LinkedIn tools
   - **Facebook**: Retrieve page insights and recent post performance using Facebook tools (page insights, post insights)

2. **Compile key metrics per platform**:
   - Total posts in the period
   - Total reach / impressions
   - Total engagements (likes, comments, shares, clicks)
   - Engagement rate (engagements / reach or impressions)
   - Follower count and growth
   - Top performing post(s) with metrics

3. **Analyze performance**:
   - Which platform is driving the most engagement?
   - What content topics or formats performed best?
   - What posting patterns correlate with higher performance?
   - Are there any concerning trends (declining reach, lower engagement)?

4. **Generate recommendations** — Provide 3-5 specific, data-driven recommendations:
   - What to do more of (based on top performers)
   - What to adjust (based on underperformers)
   - New opportunities to explore
   - Optimal posting frequency and timing insights

5. **Format the report**:
   - Executive summary (2-3 sentences)
   - Platform-by-platform breakdown with metrics tables
   - Top posts highlight
   - Content analysis insights
   - Prioritized recommendations

## Output
Present the full report in a clear, scannable format. Use tables for metrics, bold key numbers. Lead with insights, not raw data.

After presenting the report, ask if the user wants to:
- Deep-dive into any specific platform
- Export the report as a document
- Act on any of the recommendations immediately
