description: Draft responses to recent comments and mentions
---

Review recent social media interactions and draft responses for Aimersion AI across connected platforms.

## Process

1. **Gather recent interactions** — Check connected social platforms for:
   - Facebook: Recent comments on page posts (use Facebook tools to retrieve comments)
   - X: Recent mentions or replies (use X tools to search for mentions)
   - LinkedIn: Recent post engagement (use LinkedIn tools to check posts)

2. **Categorize each interaction**:
   - **Positive / Praise** — Warm thank-you with engagement follow-up
   - **Question / Support** — Helpful answer or resource link
   - **Negative / Complaint** — Empathetic acknowledgment with resolution path
   - **General mention** — Friendly acknowledgment
   - **Spam / Troll** — Flag for user review, recommend action (ignore/block/hide)

3. **Draft responses** — For each non-spam interaction, write a reply that is:
   - On-brand (knowledgeable, approachable, empowering)
   - Platform-appropriate in tone and length
   - Helpful and value-adding
   - Personalized (use the commenter's name when available)

4. **Prioritize** — Present responses in priority order: complaints first, then questions, then mentions, then praise.

5. **Present for approval** — Show each drafted response with the original message for context. Ask the user to confirm before posting any replies.

## Important
- Never post replies without explicit user approval
- Flag any interactions that may need escalation (legal issues, PR crises, sensitive topics)
- If no recent interactions are found on a platform, note it and move to the next
