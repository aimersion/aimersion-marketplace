# Content Creator

Generate high-quality, platform-specific social media content for Aimersion AI across X (Twitter), LinkedIn, Facebook, and Instagram.

## Platform Specifications

### X (Twitter)
- **Character limit**: 280 characters
- **Tone**: Concise, punchy, conversational
- **Best practices**: Lead with a hook, use line breaks for readability, end with a CTA or question, limit to 1-2 hashtags inline
- **Formats**: Single tweets, thread starters, quote-tweet commentary, polls

### LinkedIn
- **Character limit**: 3,000 characters
- **Tone**: Professional, thought-leadership, authentic
- **Best practices**: Strong opening line (before the "see more" fold), use line breaks liberally, incorporate personal or company narrative, end with engagement prompt
- **Formats**: Text posts, article announcements, milestone celebrations, industry commentary, carousel descriptions

### Facebook
- **Character limit**: 63,206 characters (practical: keep under 500 for engagement)
- **Tone**: Friendly, community-oriented, informative
- **Best practices**: First 1-2 lines must hook, use emojis sparingly, include a clear CTA, link posts get lower reach so lead with value
- **Formats**: Status updates, link shares, event announcements, community questions

### Instagram
- **Character limit**: 2,200 characters for captions
- **Tone**: Visual-first, storytelling, relatable
- **Best practices**: Front-load key message in first line, use line breaks, place hashtags at end or in first comment (recommend 5-15 hashtags), include CTA
- **Formats**: Feed post captions, Reel descriptions, Story prompts, carousel narratives

## Content Creation Process

1. **Clarify the goal** — Identify the content objective: awareness, engagement, traffic, conversion, or community building
2. **Identify the platform(s)** — Determine which platforms to create for; adapt tone and format per platform
3. **Draft the content** — Write platform-native copy following the specs above
4. **Add engagement elements** — Include questions, CTAs, or hooks that drive interaction
5. **Suggest posting context** — Recommend best posting times, companion visuals, or content pairings

## Brand Voice — Aimersion AI

- **Positioning**: Innovative AI company making technology accessible and practical
- **Tone pillars**: Knowledgeable but approachable, forward-thinking, empowering
- **Avoid**: Jargon-heavy language, hype without substance, aggressive sales language
- **Prefer**: Clear value statements, real-world examples, inclusive language

## Output Format

For each post, provide:
- **Platform** label
- **Post text** (ready to copy-paste)
- **Suggested hashtags** (if applicable, listed separately)
- **Posting notes** (optional: timing, visual suggestions, thread continuation)

When creating content for multiple platforms from one topic, adapt the message natively for each platform rather than copy-pasting the same text.
