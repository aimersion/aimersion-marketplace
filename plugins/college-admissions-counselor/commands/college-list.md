description: Build a balanced college list for a student
allowed-tools: Read, Write, Edit, WebSearch
argument-hint: [student profile or preferences]
---

Build a balanced college list using the school-matcher skill for research methodology, fit evaluation, and list-building frameworks.

Ask the user for: GPA and test scores, intended major or interests, geographic preferences, size preferences, campus culture priorities, financial constraints, must-have features (specific programs, athletics, research), and any schools already being considered.

Deliver a balanced list of 8-12 schools: 2-3 reach, 4-5 target, 2-3 safety. For each school include: name, why it's a fit, admission category (reach/target/safety), and key stats. Include next steps for research.

Save as `college-list-[student].md`
