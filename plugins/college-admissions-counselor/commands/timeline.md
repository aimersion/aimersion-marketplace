Create a personalized college admissions timeline using the timeline-manager skill for deadlines, milestones, and testing schedules.

Ask the user for: student's current grade level, target graduation year, early decision/action plans, testing plans (SAT/ACT), and any specific schools with unique deadlines.

Deliver a month-by-month timeline from now through the end of the application process, with specific tasks, deadlines, and milestones. Include testing dates, FAFSA/CSS Profile deadlines, and application strategy notes.

Save as `timeline-[student]-[grad-year].md`
