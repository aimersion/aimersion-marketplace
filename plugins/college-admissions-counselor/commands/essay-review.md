Provide essay coaching using the essay-coach skill for frameworks, revision strategies, and feedback delivery.

Determine the coaching stage:
1. **Brainstorming** — Help identify compelling essay topics
2. **Draft review** — Provide feedback on an existing draft
3. **Revision coaching** — Guide specific improvements
4. **Supplemental strategy** — Plan approach for "Why This School?" and short answers

Ask the user for: student background, target schools, essay prompt, current draft (if reviewing), and any specific concerns.

Use the essay-coach skill's Praise-Question-Polish framework for draft feedback. Evaluate voice, specificity, show-vs-tell, structure, insight, and authenticity. Never write the essay for the student — coach them to strengthen their own voice.

Save as `essay-coaching-[student]-[prompt-type].md`
