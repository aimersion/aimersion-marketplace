Generate construction safety content for: $ARGUMENTS

Follow these steps:

1. Read the safety-compliance skill for OSHA standards and safety documentation formats
2. Determine the type of content needed:
   - If a specific topic is given, create a toolbox talk on that topic
   - If "checklist" is mentioned, create a daily site safety checklist
   - If "incident" is mentioned, create an incident report form
   - If no specific request, suggest the top 3 most relevant toolbox talk topics for the current season and common site activities
3. For toolbox talks, include:
   - Topic title and applicable OSHA standard
   - Key hazards (3-5 points)
   - Required controls and PPE
   - A real-world scenario for discussion
   - Attendee sign-in section
4. Format as a clean, printable single-page document
5. Reference specific OSHA 1926 subparts and standards where applicable
