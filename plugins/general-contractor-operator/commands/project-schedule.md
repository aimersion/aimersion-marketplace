Create a construction project schedule.

Follow these steps:

1. Read the project-scheduler skill for sequencing methodology and templates
2. Ask for project details if not provided: project type, scope of work, desired start date, target completion
3. Break the project into phases and individual tasks based on the skill's standard sequences
4. Assign durations based on scope, adjusting for project complexity
5. Map dependencies — which tasks must complete before others can start
6. Identify the critical path (longest chain of dependent tasks)
7. Add inspection hold points and buffer for weather/lead times
8. Generate a formatted schedule showing: task name, duration, start/end dates, predecessors, assigned trade
9. Highlight key milestones and the projected completion date

If the user provides a specific project, tailor the schedule to their scope. If general, use the standard residential or commercial template from the references.
