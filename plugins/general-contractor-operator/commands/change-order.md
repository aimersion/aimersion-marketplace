Draft a change order for: $ARGUMENTS

Follow these steps:

1. Read the change-order-manager skill for pricing methodology and document templates
2. Ask for details if not provided: description of the change, reason, original contract amount, any subcontractor quotes
3. Categorize the change (owner request, field condition, design error, code requirement)
4. Calculate cost impact: material, labor, equipment, subcontractor, with standard markups
5. Assess schedule impact: additional days, critical path effect
6. Generate a professional change order document with:
   - CO number, date, project info
   - Detailed description and reason
   - Itemized cost breakdown with backup
   - Schedule impact assessment
   - Revised contract amount summary
   - Signature lines
7. Update the change order log if one exists, or create a new one
