description: Generate an influencer campaign performance report
argument-hint: [campaign name or details]
---

Generate a comprehensive performance report for the influencer campaign "$ARGUMENTS".

Follow the campaign-tracker skill's reporting methodology:

1. Ask the user for campaign data if not already provided:
   - Campaign dates and objectives
   - Creators involved and their deliverables
   - Budget and spend breakdown
   - Available performance metrics (impressions, engagements, clicks, conversions, revenue)

2. If this is a mid-campaign check-in, use the Weekly Check-In template
3. If this is a post-campaign analysis, use the Full Campaign Report template

4. Calculate all applicable KPIs:
   - CPM, CPE, CPC, CPA, ROAS
   - Creator-level performance ranking
   - Content type performance comparison

5. Provide actionable recommendations:
   - Which creators to re-engage or drop
   - Content themes that worked vs. didn't
   - Budget reallocation suggestions for next campaign

6. Flag any compliance or quality concerns discovered during the review
