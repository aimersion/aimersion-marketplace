Design an affiliate program structure for "$ARGUMENTS".

Follow the affiliate-program skill's methodology:

1. Ask clarifying questions about:
   - Business type (DTC, SaaS, marketplace, service)
   - Average order value and customer LTV
   - Current influencer program status (starting fresh vs. adding affiliate layer)
   - Budget constraints and target ROAS
   - Ecommerce platform (Shopify, WooCommerce, etc.)
   - Number of affiliates planned (initial and 12-month target)

2. Recommend a commission structure:
   - Choose between flat, tiered, hybrid, product seeding, or revenue share
   - Set specific commission rates with rationale based on industry benchmarks
   - Define tier thresholds if using tiered structure
   - Calculate expected economics (what does the brand pay at various volume levels?)

3. Design the program infrastructure:
   - Recommend an affiliate platform based on the comparison in references/affiliate-platforms.md
   - Define attribution windows and rules
   - Set program policies (prohibited methods, payment terms, etc.)

4. Create an onboarding plan:
   - List materials needed for creator onboarding
   - Suggest a recruitment strategy for the first 20-50 affiliates

5. Provide a 90-day launch roadmap with milestones

6. Include a program economics projection showing break-even and target scenarios
