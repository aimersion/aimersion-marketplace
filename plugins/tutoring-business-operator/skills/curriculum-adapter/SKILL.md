# Curriculum Adapter

Design differentiated instruction plans, accommodations, and scaffolding strategies that tailor tutoring to each student's unique learning profile, abilities, and needs.

## Differentiation Framework

### Four Dimensions of Differentiation
1. **Content** — What the student learns
   - Adjust reading level of materials
   - Provide multiple representations (visual, verbal, kinesthetic)
   - Offer choice in topics within the same skill area
   - Use tiered assignments at different complexity levels

2. **Process** — How the student learns
   - Vary instructional strategies (direct instruction, guided discovery, inquiry)
   - Adjust pacing (more time, compressed timeline)
   - Scaffold steps differently based on readiness
   - Use different questioning levels (Bloom's Taxonomy)

3. **Product** — How the student demonstrates learning
   - Allow multiple formats (written, oral, visual, digital)
   - Offer project choices that play to student strengths
   - Adjust rubric expectations while maintaining rigor
   - Use portfolio assessment for ongoing progress

4. **Environment** — The learning conditions
   - Adjust session location (quiet room, library, online)
   - Modify session length (shorter with breaks, or extended deep-dive)
   - Provide fidgets, standing option, or movement breaks
   - Control distractions (noise level, visual clutter)

## Accommodation Strategies by Need

### Students with ADHD/Executive Function Challenges
- Break tasks into small, concrete steps with checklists
- Use timers for focused work intervals (Pomodoro: 15-20 min for younger, 25 min for older)
- Provide written instructions alongside verbal
- Use color-coding and visual organizers
- Build in movement breaks every 15-20 minutes
- Start sessions with a brief review of goals and end with a summary
- Use high-interest materials to maintain engagement

### Students with Dyslexia/Reading Differences
- Use structured literacy approaches (Orton-Gillingham based)
- Provide text-to-speech tools and audiobooks
- Use larger fonts, increased spacing, and cream/colored backgrounds
- Teach decoding strategies explicitly
- Allow extra time for reading tasks
- Focus on comprehension strategies separately from decoding
- Use graphic organizers for writing

### English Language Learners
- Pre-teach key vocabulary before the lesson
- Use visual supports (pictures, diagrams, real objects)
- Provide bilingual glossaries for academic terms
- Simplify language without reducing content complexity
- Allow response in native language when building understanding
- Use sentence frames and paragraph templates for writing
- Connect new concepts to cultural background knowledge

### Gifted/Advanced Learners
- Compact curriculum — skip mastered material, focus on gaps and extensions
- Provide enrichment problems that require higher-order thinking
- Introduce above-grade-level concepts when ready
- Use open-ended problems with multiple solution paths
- Encourage independent research projects
- Connect to competitions, olympiads, or advanced programs
- Develop mentorship and leadership opportunities

### Students with Math Anxiety
- Start every session with something the student can do successfully
- Use manipulatives and visual models before abstract procedures
- Normalize mistakes as part of learning ("That's a useful error")
- Teach self-talk strategies ("I can't do this YET")
- Break problems into micro-steps and celebrate each one
- Avoid timed tests initially; build fluency through games
- Share stories of growth and progress to build confidence

## Scaffolding Techniques

### Gradual Release Model
1. **I Do** — Tutor models the complete skill with think-aloud
2. **We Do** — Student and tutor work through problems together
3. **You Do (supported)** — Student works independently with tutor available for hints
4. **You Do (independent)** — Student works fully independently

### Types of Scaffolds
- **Procedural**: Step-by-step checklists, worked examples, graphic organizers
- **Conceptual**: Analogies, visual models, manipulatives, real-world connections
- **Strategic**: Think-aloud protocols, problem-solving frameworks, mnemonics
- **Metacognitive**: Self-monitoring checklists, error analysis, reflection prompts

### Scaffold Fading Plan
- Start with maximum support
- Remove supports one at a time as competence grows
- Monitor for stress or regression when removing scaffolds
- Re-introduce scaffolds temporarily during new or challenging content
- Document which scaffolds the student needs and which have been faded
