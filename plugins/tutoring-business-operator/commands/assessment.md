Create a diagnostic assessment or individualized learning plan using the student-assessor skill for assessment design, skill gap analysis, and goal setting.

Determine what's needed:
1. **Diagnostic assessment** — Questions and tasks to evaluate current skill levels
2. **Learning plan** — Individualized plan based on assessment results
3. **Assessment report** — Formal report summarizing findings and recommendations
4. **Reassessment** — Progress check against previous assessment

Ask the user for:
- Student grade level and subject(s)
- Specific areas of concern or focus
- Previous assessment data (if reassessment)
- School curriculum or standards being followed
- Any known learning differences or accommodations
- Parent/student goals

For assessments, create questions that span grade levels to find the student's floor and ceiling. For learning plans, set 3-5 SMART goals with immediate wins and longer-term targets. Follow the student-assessor skill's framework for structure.

Save as `assessment-[student]-[subject]-[date].md`
