Generate client outreach communications for the veterinary practice. Type: $ARGUMENTS

If no type is specified, ask the user which they need:
- Vaccination reminders (due or overdue)
- Post-visit follow-up (wellness check, surgery, sick visit)
- Appointment reminders (upcoming visit)
- Reactivation campaign (lapsed clients)
- Seasonal health alerts (heartworm season, tick prevention, holiday hazards)
- New service announcement
- Condolence/sympathy message (pet loss)

## For Each Outreach Type, Generate:

1. **Text/SMS version** — Short, friendly, include booking link or phone number. Always use the pet's name.

2. **Email version** — Warm and personal tone. Include:
   - Pet's name in the subject line and greeting
   - Reason for outreach
   - Specific action needed
   - Easy booking CTA
   - Practice contact info

3. **Phone call script** — For the front desk team. Include:
   - Opening line
   - Key points to cover
   - Common questions/objections
   - Booking the appointment
   - Closing

## Tone Guidelines

- Always use the pet's name (and owner's name when possible)
- Warm, caring, non-judgmental
- For overdue reminders: concerned, not guilt-tripping
- For post-surgery follow-ups: reassuring, checking on the pet and the owner
- For condolence messages: genuinely empathetic, brief, offer support

## Special Handling

**Condolence/Sympathy Messages:**
- These require exceptional care and sensitivity
- Keep it personal and genuine — never templated-sounding
- Mention the pet by name and a specific positive memory if known
- Offer the practice's sympathy card or memorial options
- Do NOT include marketing, appointment reminders, or future visit language
- Wait at least 6 months before any reactivation outreach

**Euthanasia Follow-Up:**
- Send a sympathy card within 3 days
- Optional: call from the veterinarian within 1 week
- Do not send automated reminders for the deceased pet (flag in system immediately)
