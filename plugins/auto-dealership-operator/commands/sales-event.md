Plan a comprehensive dealership sales event from concept to execution.

Ask the user for:
- Event type or theme (Memorial Day sale, year-end clearance, tent sale, model launch, etc.)
- Proposed dates
- Budget range (if known)
- Specific goals (units to sell, gross profit target, new customer acquisition)
- Any inventory they specifically want to move

Generate a complete event plan with:

1. **Event Overview**: Theme, dates, goals, and success metrics
2. **Pre-Event Timeline**: Week-by-week task checklist starting 6 weeks out — advertising launches, creative needs, website updates, BDC preparation
3. **Marketing Campaign**:
   - Email sequence (save-the-date, main invite, last-chance reminder)
   - Social media content calendar for the 2 weeks before and during the event
   - Suggested ad copy for Facebook/Instagram and Google
   - Budget allocation recommendations
4. **Day-Of Operations**: Staffing plan, sales floor strategy, BDC appointment targets
5. **Post-Event Follow-Up**: Thank-you messages, unsold lead follow-up plan, ROI analysis template

Use the marketing-planner skill for event timing, creative templates, and budget benchmarks. Make every deliverable actionable — the user should be able to hand this plan to their team and execute it.
